# Projeto de Transfer Learning em Python:
O projeto consiste em criar uma rede de Deep Learning para a classificação de duas classes de imagens, gatos e cachorros, com a aplicação do método de Transfer Learning. 
<br>
<br>
<br>
## O dataset utilizado pode ser encontrado neste link:
https://www.microsoft.com/en-us/download/details.aspx?id=54765.
<br>
<br>
<br>
## O modelo reaproveitado para o Transfer Learning foi o modelo MobileNetV2, que pode ser encontrado neste link:
https://keras.io/api/applications/mobilenet/#mobilenetv2-function
